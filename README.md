# To-Do App

A Python-based To-Do App that helps users manage daily tasks efficiently.

- View Tasks
- Mark Task as Complete
- Delete Task
- File Handling Storage

## Technologies Used

- Functions
- Loops
- Conditional Statements
- File Handling
